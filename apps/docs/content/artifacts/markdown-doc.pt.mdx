## Visão geral

Um artifact de documento para quando o modelo produz prosa mais longa.
O modelo emite `{ title, body }` onde `body` é Markdown; o
artifact o renderiza dentro de um Card com tipografia `prose` afinada para
a paleta da Gremorie, incluindo dark mode via `prose-invert`.

Use este artifact para relatórios de status, resumos, notas de reunião e
qualquer output longform estruturado que o modelo gere.

## Schema

O LLM retorna output estruturado que segue este shape:

```ts
{
  type: "markdown-doc",
  title: string,
  body: string,        // CommonMark markdown
  filename?: string,
}
```

## Anatomia

```text
Card
├─ CardHeader            CardTitle + filename + Badge (outline)
└─ article.prose         typography wrapper (dark:prose-invert) over rendered Markdown
```

## Comportamento de streaming

<Callout type="info">
  Este artifact renderiza em streaming. Conforme os tokens de Markdown chegam, o
  documento cresce de cima para baixo. Headings, listas e blockquotes surgem no
  lugar conforme são parseados, sem flash entre os estados.
</Callout>
