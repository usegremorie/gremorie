## Coming soon

Artifacts are the AI-native diferencial of Gremorie: components designed to receive **structured LLM output** (parsed against a schema) and render it as UI. Each artifact ships with:

- TypeScript / Zod schema for the LLM contract
- Streaming behavior (loading parcial during generation)
- AI prompt examples (how to ask the LLM to emit valid output)
- Renderer wired with the corresponding primitives

Planned categories:

- **Chart** - bar, line, pie, area, multi-series (data viz generated by AI)
- **Code** - code-block, code-diff, code-runner (with sandbox embed)
- **Document** - markdown, outline, document-canvas
- **Data** - table, list, json-tree
- **Form** - dynamic form from JSON Schema, multi-step survey
- **Interactive** - calculator, timeline, kanban, gallery

For the AI primitives that compose artifacts, see [Components / AI](/components/ai/prompt-input).
