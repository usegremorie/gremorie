title: Dropdown Menu

`DropdownMenu` is the action menu attached to a trigger. Use it for
**actions** - Edit, Delete, Duplicate, Open in new tab. When the items
in the menu read like verbs, this is the right primitive. For values
(USA, Brazil, Japan), reach for `Select` instead.

It supports nested submenus (`DropdownMenuSub`), keyboard shortcuts
(`DropdownMenuShortcut`), checkbox and radio items, and a destructive
variant via `variant="destructive"` on individual items.

```tsx
import { Button } from "@gremorie/rx-forms";
import {
  DropdownMenu,
  DropdownMenuContent,
  DropdownMenuItem,
  DropdownMenuLabel,
  DropdownMenuSeparator,
  DropdownMenuShortcut,
  DropdownMenuTrigger,
} from "@gremorie/rx-overlays";

export function RowActions() {
  return (
    <DropdownMenu>
      <DropdownMenuTrigger asChild>
        <Button variant="ghost" size="icon-sm" aria-label="Open row actions">
          ...
        </Button>
      </DropdownMenuTrigger>
      <DropdownMenuContent align="end">
        <DropdownMenuLabel>Actions</DropdownMenuLabel>
        <DropdownMenuItem>
          Edit
          <DropdownMenuShortcut>Cmd+E</DropdownMenuShortcut>
        </DropdownMenuItem>
        <DropdownMenuItem>Duplicate</DropdownMenuItem>
        <DropdownMenuSeparator />
        <DropdownMenuItem variant="destructive">Delete</DropdownMenuItem>
      </DropdownMenuContent>
    </DropdownMenu>
  );
}
```

## Anatomy

```
DropdownMenu
  DropdownMenuTrigger (asChild = wraps a Button)
  DropdownMenuContent
    DropdownMenuLabel
    DropdownMenuItem [variant="destructive"]
    DropdownMenuCheckboxItem
    DropdownMenuRadioGroup
      DropdownMenuRadioItem
    DropdownMenuSeparator
    DropdownMenuSub
      DropdownMenuSubTrigger
      DropdownMenuSubContent
    DropdownMenuShortcut
```

## Best practices

- **Always trigger via a button.** Anchor tags break menu semantics.
- **Use `asChild` on the trigger.** Lets you wrap the project's `Button`
  primitive and inherit its styles.
- **Use `variant="destructive"` for irreversible items.** It paints the
  item with destructive tokens automatically.
- **Reach for `Select` when items hold values, not actions.**
