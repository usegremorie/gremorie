`Tool` is part of the **AI chatbot family** brought from Bridge into Gremorie in Fase 5j. It composes on top of other Gremorie primitives (`@gremorie/rx-display`, `@gremorie/rx-overlays`, `@gremorie/rx-forms`, ...) and ships with full Bridge parity.

## Install

<Tabs items={["Angular", "React"]}>

<Tab value="Angular">

<Tab value="Angular">

Em breve.

<Tab value="React">

```tsx
import {
  Tool,
  ToolHeader,
  ToolContent,
  ToolInput,
  ToolOutput,
} from '@gremorie/rx-ai';

export function Demo() {
  return (
    <Tool defaultOpen>
      <ToolHeader type="tool-search" state="output-available" />
      <ToolContent>
        <ToolInput input={params} />
        <ToolOutput output={result} errorText={error} />
      </ToolContent>
    </Tool>
  );
}
```

## Notes

- This primitive was migrated from Bridge into `@gremorie/rx-ai` as part of the Fase 5j chatbot batch (Bridge parity for the AI category).
- All cross-Gremorie imports use the published `@gremorie/rx-*` packages; the registry mirrors that into per-primitive source so consumers can `npx gremorie add rx-tool` without pulling unrelated AI components.
