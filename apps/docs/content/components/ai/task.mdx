---
title: Task
description: Collapsible item list for an agent task. TaskTrigger shows status + summary, TaskContent holds TaskItem children with inline file badges.
---

`Task` is part of the **AI chatbot family** brought from Bridge into Gremorie in Fase 5j. It composes on top of other Gremorie primitives (`@gremorie/rx-display`, `@gremorie/rx-overlays`, `@gremorie/rx-forms`, ...) and ships with full Bridge parity.

## Usage

<Tabs items={["Angular", "React"]}>

<Tab value="Angular">

Em breve.

</Tab>

<Tab value="React">

```tsx
import {
  Task,
  TaskTrigger,
  TaskContent,
  TaskItem,
  TaskItemFile,
} from '@gremorie/rx-ai';

export function Demo() {
  return (
    <Task>
      <TaskTrigger title="Refactor router" />
      <TaskContent>
        <TaskItem>Edit <TaskItemFile>app.tsx</TaskItemFile></TaskItem>
      </TaskContent>
    </Task>
  );
}
```

## Notes

- This primitive was migrated from Bridge into `@gremorie/rx-ai` as part of the Fase 5j chatbot batch (Bridge parity for the AI category).
- All cross-Gremorie imports use the published `@gremorie/rx-*` packages; the registry mirrors that into per-primitive source so consumers can `npx gremorie add rx-task` without pulling unrelated AI components.
