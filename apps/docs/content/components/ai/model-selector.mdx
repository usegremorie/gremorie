title: Model Selector
description: Floating model picker dialog - Cmd+K-style with a full Command palette inside a Dialog.
---

`Model Selector` is part of the **AI chatbot family** brought from Bridge into Gremorie in Fase 5j. It composes on top of other Gremorie primitives (`@gremorie/rx-display`, `@gremorie/rx-overlays`, `@gremorie/rx-forms`, ...) and ships with full Bridge parity.

## Install

<Tabs items={["Angular", "React"]}>

<Tab value="Angular">

Em breve. A categoria AI no Angular ainda so cobre PromptInput. Use a edicao React por enquanto, ou abra uma issue para priorizar este primitivo.

</Tab>

<Tab value="React">

```bash
npx gremorie add rx-model-selector
```

Pulls in the registry dependencies declared for `rx-model-selector` (cross-primitive Gremorie components - see `registry.json`).

</Tab>

## Usage

<Tabs items={["Angular", "React"]}>

<Tab value="Angular">

Em breve.

</Tab>

<Tab value="React">

```tsx
import {
  ModelSelector,
  ModelSelectorTrigger,
  ModelSelectorContent,
  ModelSelectorCommand,
  ModelSelectorInput,
  ModelSelectorList,
  ModelSelectorEmpty,
  ModelSelectorGroup,
  ModelSelectorItem,
  ModelSelectorSeparator,
  ModelSelectorShortcut,
} from '@gremorie/rx-ai';

export function Demo() {
  return (
    <ModelSelector>
      <ModelSelectorTrigger>{currentModel}</ModelSelectorTrigger>
      <ModelSelectorContent title="Choose a model">
        <ModelSelectorInput placeholder="Search..." />
        <ModelSelectorList>
          <ModelSelectorGroup heading="Frontier">
            <ModelSelectorItem value="opus-4">Opus 4</ModelSelectorItem>
          </ModelSelectorGroup>
        </ModelSelectorList>
      </ModelSelectorContent>
    </ModelSelector>
  );
}
```

## Notes

- This primitive was migrated from Bridge into `@gremorie/rx-ai` as part of the Fase 5j chatbot batch (Bridge parity for the AI category).
- All cross-Gremorie imports use the published `@gremorie/rx-*` packages; the registry mirrors that into per-primitive source so consumers can `npx gremorie add rx-model-selector` without pulling unrelated AI components.
