`Open in Chat` is part of the **AI utilities family** brought from Bridge into Gremorie in Fase 5j.

## Usage

<Tabs items={["Angular", "React"]}>

<Tab value="Angular">

Em breve.

</Tab>

<Tab value="React">

```tsx
import {
  OpenInChat,
  OpenInChatTrigger,
  OpenInChatContent,
  OpenInChatLabel,
  OpenInChatSeparator,
  OpenInChatItem,
} from '@gremorie/rx-ai';

export function Demo() {
  return (
    <OpenInChat url={pageUrl}>
      <OpenInChatTrigger />
      <OpenInChatContent>
        <OpenInChatLabel>Open in...</OpenInChatLabel>
        <OpenInChatSeparator />
        <OpenInChatItem provider="chatgpt" />
        <OpenInChatItem provider="claude" />
      </OpenInChatContent>
    </OpenInChat>
  );
}
```
