`Checkpoint` is part of the **AI chatbot family** brought from Bridge into Gremorie in Fase 5j. It composes on top of other Gremorie primitives (`@gremorie/rx-display`, `@gremorie/rx-overlays`, `@gremorie/rx-forms`, ...) and ships with full Bridge parity.

import {
  Checkpoint,
  CheckpointIcon,
  CheckpointLabel,
  CheckpointActions,
  CheckpointAction,
} from '@gremorie/rx-ai';

export function Demo() {
  return (
    <Checkpoint>
      <CheckpointIcon />
      <CheckpointLabel>Saved at 14:32</CheckpointLabel>
      <CheckpointActions>
        <CheckpointAction tooltip="Revert" onClick={revert}>Revert</CheckpointAction>
      </CheckpointActions>

## Notes

- This primitive was migrated from Bridge into `@gremorie/rx-ai` as part of the Fase 5j chatbot batch (Bridge parity for the AI category).
- All cross-Gremorie imports use the published `@gremorie/rx-*` packages; the registry mirrors that into per-primitive source so consumers can `npx gremorie add rx-checkpoint` without pulling unrelated AI components.
