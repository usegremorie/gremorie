---
title: Edge
description: Edge variants for @xyflow/react. Temporary (simple-bezier, dashed) and Animated (gradient-flowing) bezier curves. Register via edgeTypes on Canvas.
---

`Edge` is part of the **AI workflow family** brought from Bridge into Gremorie in Fase 5j. Built on top of `@xyflow/react` for graph-style canvases.

## Install

<Tabs items={["Angular", "React"]}>

</Tab>

<Tab value="React">


```tsx
import {
  Temporary,
  Animated,
} from '@gremorie/rx-ai';

export function Demo() {
  return (
    const edgeTypes = { temporary: Temporary, animated: Animated };
    
    <Canvas
```

## Notes

- This primitive was migrated from Bridge into `@gremorie/rx-ai` as part of the Fase 5j workflow batch (Bridge parity for the AI category).
- The workflow family wraps `@xyflow/react` with Gremorie theming - keep `@xyflow/react/dist/style.css` imported once at the app level.
