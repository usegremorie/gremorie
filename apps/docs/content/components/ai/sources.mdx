---
title: Sources
description: Citations cluster for RAG responses. SourcesTrigger toggles, SourcesContent lists Source hyperlinks with optional favicons.
---

`Sources` is part of the **AI chatbot family** brought from Bridge into Gremorie in Fase 5j. It composes on top of other Gremorie primitives (`@gremorie/rx-display`, `@gremorie/rx-overlays`, `@gremorie/rx-forms`, ...) and ships with full Bridge parity.

## Install

<Tabs items={["Angular", "React"]}>

<Tab value="Angular">

## Usage

<Tabs items={["Angular", "React"]}>

<Tab value="Angular">

Em breve.

</Tab>

<Tab value="React">

```tsx
import {
  Sources,
  SourcesTrigger,
  SourcesContent,
  Source,
} from '@gremorie/rx-ai';

export function Demo() {
  return (
    <Sources>
      <SourcesTrigger count={3} />
      <SourcesContent>
        <Source href="https://docs.example.com" title="Docs" />
        <Source href="https://blog.example.com" title="Blog" />
      </SourcesContent>
    </Sources>
  );
}
```

## Notes

- This primitive was migrated from Bridge into `@gremorie/rx-ai` as part of the Fase 5j chatbot batch (Bridge parity for the AI category).
- All cross-Gremorie imports use the published `@gremorie/rx-*` packages; the registry mirrors that into per-primitive source so consumers can `npx gremorie add rx-sources` without pulling unrelated AI components.
