## Usage

<Tabs items={["Angular", "React"]}>

<Tab value="Angular">

Em breve.

</Tab>

<Tab value="React">

```tsx
import {
  WebPreview,
  WebPreviewNavigation,
  WebPreviewNavigationButton,
  WebPreviewUrl,
  WebPreviewBody,
  WebPreviewConsole,
  WebPreviewConsoleTrigger,
  WebPreviewConsoleContent,
} from '@gremorie/rx-ai';

export function Demo() {
  return (
    <WebPreview defaultUrl="https://example.com">
      <WebPreviewNavigation>
        <WebPreviewUrl />
      </WebPreviewNavigation>
      <WebPreviewBody />
      <WebPreviewConsole>
        <WebPreviewConsoleTrigger />
        <WebPreviewConsoleContent logs={consoleLogs} />
      </WebPreviewConsole>
    </WebPreview>
  );
}
```

</Tab>

</Tabs>

## Notes

- This primitive was migrated from Bridge into `@gremorie/rx-ai` as part of the Fase 5j code batch (Bridge parity for the AI category).
- All cross-Gremorie imports use the published `@gremorie/rx-*` packages; the registry mirrors that into per-primitive source so consumers can `npx gremorie add rx-web-preview` without pulling unrelated AI components.
