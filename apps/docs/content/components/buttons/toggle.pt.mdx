## Acessibilidade

- **ARIA**: o Radix renderiza `role="button"` com `aria-pressed` refletindo o estado (`true` / `false`).
- **Teclado**: `Space` e `Enter` alternam o button; `Tab` / `Shift+Tab` movem o foco.
- **Focus**: focus-visible ring de 3px controlado por `focus-visible:ring-ring/50`.
- **Icon-only**: sempre forneĂ§a `aria-label`. Sem ele, leitores de tela anunciam o toggle sem nome.
- **DistinĂ§ĂŁo de Switch / Checkbox**: Toggle Ă© para **aĂ§Ăµes com estado** (aplicar um formato, mudar uma visualizaĂ§ĂŁo). Switch e Checkbox sĂŁo para **valores com estado** (uma configuraĂ§ĂŁo, um campo de formulĂˇrio). O tratamento visual difere porque o modelo mental difere.
