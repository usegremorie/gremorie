`RadarChart` plots one closed polygon per series, with each data row
becoming a spoke. Pass tabular `data`, a JSON-serializable `config`
mapping each value field to a `{ label, color }`, and the `xKey` that
names the spoke label field. Built on the shared chart engine
(`chart-context`, `chart-frame`, `radar`).

## Usage

<Tabs items={["Angular", "React"]}>

<Tab value="Angular">

```ts
import { Component } from "@angular/core";
import { RadarChart } from "@gremorie/ng-data";
import type { ChartConfig, Datum } from "@gremorie/ng-data";

@Component({
  selector: "skills-radar",
  imports: [RadarChart],
  template: `
    <radar-chart [data]="data" [config]="config" xKey="skill" />
  `,
})
export class SkillsRadar {
  readonly data: Datum[] = [
    { skill: "Design", alice: 80, bob: 60 },
    { skill: "Code", alice: 65, bob: 90 },
    { skill: "Speak", alice: 70, bob: 75 },
    { skill: "Lead", alice: 60, bob: 70 },
  ];
  readonly config: ChartConfig = {
    alice: { label: "Alice", color: "var(--chart-1)" },
    bob: { label: "Bob", color: "var(--chart-2)" },
  };
}
```

</Tab>

<Tab value="React">

```tsx
import { RadarChart } from "@gremorie/rx-data";
import type { ChartConfig, Datum } from "@gremorie/rx-data";

const data: Datum[] = [
  { skill: "Design", alice: 80, bob: 60 },
  { skill: "Code", alice: 65, bob: 90 },
  { skill: "Speak", alice: 70, bob: 75 },
  { skill: "Lead", alice: 60, bob: 70 },
];

const config: ChartConfig = {
  alice: { label: "Alice", color: "var(--chart-1)" },
  bob: { label: "Bob", color: "var(--chart-2)" },
};

export function SkillsRadar() {
  return <RadarChart data={data} config={config} xKey="skill" />;
}
```

</Tab>

## Anatomy

```
RadarChart (figure, aspect-square)
  ChartFrame (svg)
    Radar (one closed polygon per series)
    Shared rings + axes (drawn once, by the first series)
  Legend (ul)
  Table (sr-only data table)
```

## Best practices

- **Three to seven spokes.** Fewer than three is meaningless; more than
  seven gets cramped.
- **Two or three series.** Overlapping polygons lose detail fast.
- **Shared scale only.** All series share one radial maximum - if the
  series live in different units, split into separate charts.
