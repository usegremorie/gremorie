`ScatterChart` plots points against a linear numeric X axis and one or
more Y series. Unlike Area / Line / Bar, `xKey` here is a numeric field
(not a category). Built on the shared chart engine (`chart-context`,
`chart-frame`, `axis`, `cartesian-grid`, `scatter`).

## Usage

<Tabs items={["Angular", "React"]}>

<Tab value="Angular">

```ts
import { Component } from "@angular/core";
import { ScatterChart } from "@gremorie/ng-data";
import type { ChartConfig, Datum } from "@gremorie/ng-data";

@Component({
  selector: "weight-score",
  imports: [ScatterChart],
  template: `
    <scatter-chart [data]="data" [config]="config" xKey="weight" />
  `,
})
export class WeightScore {
  readonly data: Datum[] = [
    { weight: 65, score: 72 },
    { weight: 78, score: 81 },
    { weight: 90, score: 65 },
  ];
  readonly config: ChartConfig = {
    score: { label: "Score", color: "var(--chart-1)" },
  };
}
```

</Tab>

<Tab value="React">

```tsx
import { ScatterChart } from "@gremorie/rx-data";
import type { ChartConfig, Datum } from "@gremorie/rx-data";

const data: Datum[] = [
  { weight: 65, score: 72 },
  { weight: 78, score: 81 },
  { weight: 90, score: 65 },
];

const config: ChartConfig = {
  score: { label: "Score", color: "var(--chart-1)" },
};

export function WeightScore() {
  return <ScatterChart data={data} config={config} xKey="weight" />;
}
```

## Anatomy

```
ScatterChart (figure)
  ChartFrame (svg)
    CartesianGrid (horizontal gridlines)
    Scatter (one per Y series, circles per row)
    XAxis labels (min / mid / max numeric ticks)
    YAxis (value ticks)
  Table (sr-only data table)
```

## Best practices

- **Numeric X field.** `xKey` must be numeric; for categorical X use
  Line, Area, or Bar instead.
- **One or two Y series.** More than two stacks overlap and lose signal.
- **Outliers belong in the chart.** Do not pre-trim the data - the
  whole point of scatter is to see the spread.
