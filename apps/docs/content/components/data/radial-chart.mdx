---
title: Radial Chart
description: "Concentric radial bars - one ring per data row."
---

`RadialChart` plots one concentric ring per data row, each sweep
proportional to its `valueKey` value. Use `nameKey` to label each ring.
Colors cycle through `--chart-1..5`. Built on the shared chart engine
(`chart-context`, `chart-frame`, `radial-bar`).

## Usage

<Tabs items={["Angular", "React"]}>

<Tab value="Angular">

```ts
import { Component } from "@angular/core";
import { RadialChart } from "@gremorie/ng-data";
import type { Datum } from "@gremorie/ng-data";

@Component({
  selector: "browsers-radial",
  imports: [RadialChart],
  template: `
    <radial-chart [data]="data" nameKey="browser" valueKey="visitors" />
  `,
})
export class BrowsersRadial {
  readonly data: Datum[] = [
    { browser: "Chrome", visitors: 275 },
    { browser: "Safari", visitors: 200 },
    { browser: "Firefox", visitors: 187 },
    { browser: "Edge", visitors: 140 },
  ];
}
```

</Tab>

<Tab value="React">

```tsx
import { RadialChart } from "@gremorie/rx-data";

  { browser: "Firefox", visitors: 187 },

export function BrowsersRadial() {
  return (
    <RadialChart data={data} nameKey="browser" valueKey="visitors" />
```

## Anatomy

```
RadialChart (figure, aspect-square)
  ChartFrame (svg)
    RadialBar (concentric rings - track + value arc per row)
  Legend (ul)
  Table (sr-only data table)
```

## Best practices

- **Sort descending.** The component does not sort - put the largest
  value first so the outer ring carries the most weight.
- **Three to six rings.** Beyond that the bands get thin and unreadable.
- **Compare against a known max.** Radial bars communicate share, not
  absolute counts - pair with a label or sr-only table if exact values
  matter.
