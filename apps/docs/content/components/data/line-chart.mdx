title: Line Chart
description: "Trends over a continuous domain."
---

`LineChart` plots one or more series as stroked lines over a categorical
X axis. Pass tabular `data`, a JSON-serializable `config` mapping each
value field to a `{ label, color }`, and the `xKey` that names the
category field. Built on the shared chart engine (`chart-context`,
`chart-frame`, `axis`, `cartesian-grid`, `line`).

## Usage

<Tabs items={["Angular", "React"]}>

<Tab value="Angular">

```ts
import { Component } from "@angular/core";
import { LineChart } from "@gremorie/ng-data";
import type { ChartConfig, Datum } from "@gremorie/ng-data";

@Component({
  selector: "sessions-line",
  imports: [LineChart],
  template: `
    <line-chart [data]="data" [config]="config" xKey="day" />
  `,
})
export class SessionsLine {
  readonly data: Datum[] = [
    { day: "Mon", visitors: 120 },
    { day: "Tue", visitors: 200 },
    { day: "Wed", visitors: 160 },
  ];
  readonly config: ChartConfig = {
    visitors: { label: "Visitors", color: "var(--chart-1)" },
  };
}
```

</Tab>

<Tab value="React">

```tsx
import { LineChart } from "@gremorie/rx-data";
import type { ChartConfig, Datum } from "@gremorie/rx-data";

const data: Datum[] = [
  { day: "Mon", visitors: 120 },
  { day: "Tue", visitors: 200 },
  { day: "Wed", visitors: 160 },
];

const config: ChartConfig = {
  visitors: { label: "Visitors", color: "var(--chart-1)" },
};

export function SessionsLine() {
  return <LineChart data={data} config={config} xKey="day" />;
}
```

</Tab>

## Anatomy

```
LineChart (figure)
  ChartFrame (svg)
    CartesianGrid (horizontal gridlines)
    Line (one per series)
    XAxis (category ticks)
    YAxis (value ticks)
  Table (sr-only data table)
```

## Best practices

- **Prefer line over area for many series.** Filled areas overlap and
  occlude; lines stack cleanly.
- **Match colors to legend tokens.** Stick with `var(--chart-1..5)` so
  the chart matches the theme.
- **One Y unit per chart.** Mixing revenue and rate on the same axis
  hides the smaller of the two - split into two charts.
