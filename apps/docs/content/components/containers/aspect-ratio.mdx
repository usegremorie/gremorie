`AspectRatio` is a thin wrapper over Radix AspectRatio with a single
`ratio` prop. Use it on every responsive image, iframe, video embed, or
skeleton placeholder so the layout does not jump when the asset
settles - prevents cumulative layout shift (CLS) hits.

## Anatomy

```
AspectRatio [ratio={number}]
  <child />            (image, iframe, video, skeleton)
```

## Best practices

- **Always set `ratio` explicitly.** Common values: `16 / 9` for video,
  `4 / 3` for landscape, `1` for square, `3 / 4` for portrait posters.
- **Pair with `object-cover` on `<img>`.** Without it the image will not
  fill the reserved box on assets whose intrinsic ratio differs.
- **Use for skeletons too.** A `Skeleton` inside an `AspectRatio`
  reserves the same space the image will occupy, preventing the shift
  when the real asset loads.
