---
title: Progress
description: Determinate progress indicator (0-100).
---

`Progress` shows progress against a known total - uploads, multi-step
forms, batch jobs. For unknown duration, prefer `Skeleton` or a
spinner. Always pair the bar with a numeric value or label; silent
bars confuse users about the actual state.

## API

- `value` (0-100) - the current progress. Required for determinate.

## Best practices

- **Always show the numeric value.** A bare bar leaves users guessing -
  pair it with a label like `42%` or `3 of 8`.
- **Use Skeleton for unknown duration.** Progress should only appear
  when you can compute percent complete.
