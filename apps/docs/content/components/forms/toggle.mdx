`Toggle` is built on Radix Toggle - a press-button that can be on
or off. Use for inline formatting controls (bold, italic, underline)
or visibility toggles (show/hide details). For multi-option clusters
use `ToggleGroup`; for binary settings outside an editor surface use
`Switch`.
