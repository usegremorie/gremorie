`Button` is the canonical click target. It exposes six visual variants
(`default`, `destructive`, `outline`, `secondary`, `ghost`, `link`) and
eight size presets (default, `xs`, `sm`, `lg`, plus icon-only `icon`,
`icon-xs`, `icon-sm`, `icon-lg`). Pass `asChild` to render through Radix
`Slot.Root` and inherit the trigger from a child element (e.g. a Next.js
`<Link>`).

## Usage

<Tabs items={["Angular", "React"]}>

<Tab value="Angular">

```ts
import { Button } from "@gremorie/ng-core";

@Component({
  selector: "app-toolbar",
  standalone: true,
  imports: [Button],
  template: `
    <button gremorie-button variant="default">Save</button>
    <button gremorie-button variant="outline" size="sm">Cancel</button>
  `,
})
export class ToolbarComponent {}
```

## Variants

| Variant       | Use                                     |
| ------------- | --------------------------------------- |
| `default`     | Primary action - the recommended path.  |
| `destructive` | Irreversible actions (delete, revoke).  |
| `outline`     | Secondary action with visual weight.    |
| `secondary`   | Quiet secondary action.                 |
| `ghost`       | Tertiary action - toolbar buttons.      |
| `link`        | Inline action that reads like text.     |

## Sizes

| Size       | Height | Notes                              |
| ---------- | ------ | ---------------------------------- |
| `xs`       | 24px   | Compact toolbar buttons.           |
| `sm`       | 32px   | Form controls, inline actions.     |
| `default`  | 36px   | Standard size.                     |
| `lg`       | 40px   | Hero CTAs.                         |
| `icon-xs`  | 24px   | Icon-only, square.                 |
| `icon-sm`  | 32px   | Icon-only, square.                 |
| `icon`     | 36px   | Icon-only, square.                 |
| `icon-lg`  | 40px   | Icon-only, square.                 |

## Best practices

- **Use variants, not className overrides.** Reach for `variant="ghost"`
  before painting a button with one-off Tailwind classes.
- **Always pair icon-only buttons with `aria-label`.** Without text,
  screen readers and tooltips both need the label.
- **`asChild` swaps the root.** When wrapping a `<Link>`, the link
  inherits the button styles. Don't nest a button inside a button.
