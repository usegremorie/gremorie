`Slider` is built on Radix Slider - single value or range (two
thumbs), with `step`, `min`, `max`. Use for continuous values:
volume, opacity, price range, brightness.

## Usage

<Tabs items={["Angular", "React"]}>

<Tab value="Angular">

> Em breve. The Angular edition will ship in a follow-up release.

</Tab>

<Tab value="React">

```tsx
import { Slider } from "@gremorie/rx-forms";

export function VolumeSlider() {
  return <Slider defaultValue={[50]} max={100} step={1} />;
}

export function PriceRange() {
  return (
    <Slider defaultValue={[20, 80]} min={0} max={100} step={5} />
  );
}
```
