---
title: Checkbox
description: Binary or tri-state checkbox.
---

`Checkbox` is built on Radix Checkbox - binary or tri-state
(`indeterminate`). Use for multi-select lists, terms acceptance,
settings toggles. For mutually exclusive choices use `RadioGroup`;
for instant on/off use `Switch`.
