## Overview

A focused empty-state component: a soft muted background framed with
a dashed border, a circular icon medallion, a headline, a short
description, and a primary plus secondary action.

Use this block whenever a list, table, or board has no items yet.
Replace the icon, copy, and actions to match the surface.

## Customization

- Swap the icon and copy to fit context: no results (Search), no data (Database), permission denied (Lock), coming soon (Clock)
- Drop one of the buttons for a single-CTA variant
- Wrap in `Card` when the empty state needs to read as a contained section
- Add an inline illustration above the medallion for higher-fidelity surfaces
