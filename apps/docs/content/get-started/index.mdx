**Gremorie** is the AI-native design system for React and Angular:
[83 components](/components) in two editions with one shared
contract, a [three-layer token system](/tokens), and six brand
themes, all documented in one place.

There are two ways in. Pick yours.

## The developer path, step by step

1. **[Installation](/get-started/installation)**: pick registry
   (you own the source) or npm (versioned package) and add Gremorie
   to a React or Angular project.
2. **[Your first component](/get-started/first-component)**: wire
   `PromptInput`, `Conversation`, and `Message` into a working chat
   surface.
3. **[Project setup](/get-started/project-setup)**: Tailwind v4
   config, the theme cascade, path aliases, and dark mode.

From there, browse the [component catalog](/components) or go
straight to the [tokens](/tokens) that theme it.
