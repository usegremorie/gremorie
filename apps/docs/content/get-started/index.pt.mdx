O **Gremorie** é o design system AI-native para React e Angular:
[83 componentes](/components) em duas edições com um contrato
compartilhado, um [sistema de tokens em três camadas](/tokens) e
seis temas de marca, tudo documentado num lugar só.

Existem dois caminhos de entrada. Escolha o seu.

## O caminho do desenvolvedor, passo a passo

1. **[Installation](/get-started/installation)**: escolha registry
   (você é dono do código) ou npm (pacote versionado) e adicione o
   Gremorie a um projeto React ou Angular.
2. **[Your first component](/get-started/first-component)**: conecte
   `PromptInput`, `Conversation` e `Message` numa superfície de chat
   funcional.
3. **[Project setup](/get-started/project-setup)**: config do
   Tailwind v4, a cascata de tema, path aliases e dark mode.

A partir daí, navegue o [catálogo de componentes](/components) ou vá
direto para os [tokens](/tokens) que o tematizam.
