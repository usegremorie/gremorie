## Coming soon

The Gremorie CLI scaffolds registry items into your Angular or React project:

```bash
npx gremorie add rx-button
npx gremorie add ng-prompt-input
```

Features planned:

- Framework auto-detect (from package.json deps)
- Resolved dependencies (cross-item registryDependencies)
- Target path detection (e.g., `src/components/gremorie/forms/`)
- Diff preview before write

The CLI source lives in [packages/cli](https://github.com/usegremorie/gremorie/tree/develop/packages/cli) on GitHub.
